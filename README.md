# Consultancy Invoice Generator (India)

A simple, secure, and standalone web application for generating TDS-compliant consultancy invoices.

**Features:**
- 100% Client-Side (No data saved to any server)
- Indian Income Tax (TDS) logic built-in (Section 194J, 194C, etc.)
- Auto-validates PAN format and Type (Individual vs Company)
- Professional A4 PDF export
